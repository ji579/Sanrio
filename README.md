# Sanrio

프로젝트 개요
Sanrio Korea 공식 웹사이트를 참고해

메인 메뉴 구조와 전체적인 화면 구성을 분석하고,

HTML과 CSS를 중심으로 구현한 웹 퍼블리싱 프로젝트입니다.

공식 사이트는 다중 페이지 구조로 운영되고 있지만,

본 프로젝트에서는 레이아웃과 네비게이션 구조에 집중해

단일 페이지 형태로 제작했습니다.


이를 통해 브랜드의 전반적인 톤앤매너를 파악하고,

시맨틱 마크업과 CSS 레이아웃 구현 경험을 쌓았습니다.

실현 기능

	•	Sanrio 캐릭터 콘셉트에 맞춘 이미지 중심 레이아웃 구현

	•	Flexbox를 활용한 콘텐츠 정렬 및 구조 설계

	•	CSS hover 효과를 통한 버튼·이미지 인터랙션 구현

	•	JavaScript를 활용한 간단한 동적 요소 제어

	•	반복되는 UI 요소를 클래스 단위로 관리해 코드 효율성 확보

	•	전체 페이지 톤앤매너를 고려한 스타일 통일
    

사용 기술

HTML5
시맨틱 구조로 페이지별 역할을 명확히 구분하고,
작가/전시/정보 페이지를 의미 기반으로 설계
CSS3
Flexbox/그리드 레이아웃으로 구성하며, 쿠사마 작품의 감성과 조화를 이루도록
이미지 박스, 텍스트 비율, 색상·여백 등을 세밀하게 조절
JavaScript (ES6)
페이지 내 이미지/인터랙션 제어, 모션 요소, 동적 스타일 적용 등
사용자 체험 요소 구현
